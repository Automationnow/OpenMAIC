# CALE Foundation — OpenMAIC Classroom Prompts

These prompts are designed for use with the **OpenMAIC Multi-Agent Interactive Classroom** as part of the **Certified Automation Literacy Educator (CALE) Foundation** Train-the-Trainer program by **Automation Now, LLC**.

Each prompt leverages OpenMAIC's multi-agent capabilities (Facilitator, Advisor, Peer) to drive class participation, debate, and experiential learning.

---

## Prompt Index

| # | Prompt | CALE Module | Objective |
|---|---|---|---|
| 1 | [Automation Spectrum Diagnostic](./prompt-1-automation-spectrum-diagnostic.md) | Module 1 | Distinguish between the four tiers of automation using real-world processes |
| 2 | [Ethical Dilemma Simulation](./prompt-2-ethical-dilemma-simulation.md) | Module 5 | Practice facilitating difficult conversations about job displacement and human-augmentation |
| 3 | [Socratic Questioning Drill](./prompt-3-socratic-questioning-drill.md) | Day 2 — Station 1 | Build muscle memory for open-ended, clarifying, and challenging questions |
| 4 | [Process Scorecard Challenge](./prompt-4-process-scorecard-challenge.md) | Module 2 | Test ability to accurately score and debate process feasibility |

## How to Use

1. Navigate to [open-maic-zeta-three.vercel.app](https://open-maic-zeta-three.vercel.app)
2. Copy the prompt text from the relevant file below
3. Paste it into the OpenMAIC text input box on the homepage
4. Click **Enter Classroom** — the multi-agent session will begin

> **Note:** Ensure Mistral Voxtral TTS is enabled in Settings → Text-to-Speech for voice output using Derrick Voice 3.
